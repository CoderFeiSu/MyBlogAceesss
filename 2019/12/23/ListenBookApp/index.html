分析登录协议

请求参数

  • 到登录界面，输入用户名和密码，点击登录。通过Charles抓包如图
  • 上图可知请求分为三部分:
    • BaseURL:https://XXOO.info/XXOO/ClientLogon.action
    • Query String: imei和token
    • Form: account,pwd,nwt,q,sc
  • Query String拼接到BaseURL后面，Form就是Body内容

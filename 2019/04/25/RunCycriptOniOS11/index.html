在iOS11上面运行cycript

发表于 2019-04-25 更新于 2021-05-11 分类于 iOS逆向

如果使用其它越狱工具请清除越狱环境,然后使用Undecimus3.0.0以上版本越狱。

  • 添加软件源 apt.saurik.com/cydia
  • 通过Cydia安装cycript
  • 安装后从/usr/bin中复制cycript到本地,
  • 通过ldid命令导出文件以前的签名权限ldid -e cycript > cycript.entitlements
  • 使用Xcode打开cycript.entitlements添加platform-application 设置为YES.
  • 过ldid命令重签名 ldid -Scycript.entitlements cycript
  • 将已经签好权限的cycript放到iPhone上/usr/bin覆盖原来的cycript
